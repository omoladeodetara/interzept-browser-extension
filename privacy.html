Data Collection and Usage

The Interzept Chrome Extension is designed with privacy as a core principle. We do not collect, store, or transmit any personal data or browsing information. This extension serves solely as a launcher tool to provide quick access to the Interzept web application.

What we DON'T do:
  • Collect personal information
  • Track browsing behavior
  • Store user data locally or remotely
  • Use analytics or tracking services
  • Access or read webpage content
  • Monitor user activity

Permissions Justification

Tabs Permission

The extension requests the "tabs" permission for one specific purpose:

  • Primary Function: Open the Interzept web application (https://app.interzept.com) in a new browser tab
  • Implementation: Uses the chrome.tabs.create() API to launch the application
The extension does NOT:
  • Read content from any tabs
  • Monitor browsing behavior across tabs
  • Access or modify webpage data
  • Track user activity across different websites
  • Inject scripts into webpages

Host Permissions

The extension may request host permissions to facilitate seamless integration with the Interzept web application and enable direct navigation to the platform.

Sole Purpose

The Interzept Chrome Extension has a single, clearly defined purpose:

To provide developers with quick and convenient access to the Interzept web application for API testing and debugging.

The extension functions as a simple launcher that:

  1. Displays a popup interface when the extension icon is clicked
  2. Opens the Interzept web application (https://app.interzept.com) in a new browser tab
  3. Provides no additional functionality beyond this navigation feature

Remote Code

This extension does not execute any remote code. All functionality is contained within the extension package itself:

  • No external scripts are loaded or executed
  • No remote resources are fetched for functionality purposes
  • All code is static and included in the extension bundle
  • No use of eval() or similar dynamic code execution

Data Storage

The extension maintains its commitment to privacy by not storing any user data:

  • No cookies: The extension does not set or access any cookies
  • No local storage: No user data is stored locally on the device
  • No remote storage: No data is transmitted to external servers
  • No analytics: No usage tracking or analytics data is collected

Third-Party Services

The extension only interacts with the official Interzept web application at https://app.interzept.com. No other third-party services are contacted, used, or integrated with this extension.

Security

We implement security best practices to ensure user safety:

  • Content Security Policy (CSP) restrictions
  • Minimal permission requests
  • No remote code execution
  • Secure manifest v3 implementation
